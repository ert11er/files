# Project Brief: Turkish Lesson Notes Blog

## Goal

Create a personal blog for hosting "lesson notes" (ders notları) in Turkish, hosted on GitHub Pages at `/blog/`.

## Core Requirements

- **Language**: Turkish (Türkçe).
- **Hosting**: GitHub Pages (`/blog/` subpath).
- **Format**: Lesson notes, personal feel.
- **SSG-like**: Posts are `.md` files in a folder.

## Key Features

- RSS Feed
- Tagging system
- Search functionality
- Dark/Light mode toggle
- Metadata/Navigation starting from `/blog/`

## Constraints

- Must handle the `/blog/` path correctly in all internal links.
- No heavy backend; must be static-site friendly.
- Desktop and Mobile responsive.
