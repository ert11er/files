# Product Context

## Why this project exists?

The user wants a dedicated, personal space to share Turkish lesson notes. The goal is to make these notes easily accessible, searchable, and organized by tags.

## Problems it solves

- Disorganized notes.
- Hard to find specific topics within a collection of files.
- Lack of a "blog-like" experience for personal learning material.

## How it should work

- Users (the author) drop `.md` files into a `posts/` folder.
- The site automatically indexes these files via `posts.json`.
- Visitors can browse the main feed, search for keywords, or filter by tags (dersler/konular).
- Readers can toggle between dark and light modes for comfortable reading.

## User Experience Goals

- **Personal & Educational**: The UI should feel like a well-organized notebook or a modern educational blog.
- **Fast & Lightweight**: Minimal loading times.
- **Native Turkish Experience**: All UI elements (tabs, buttons, search placeholders) are in Turkish.
